# Task 2: Exploratory Data Analysis (EDA)

## Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on the Titanic dataset using Python and visualization libraries to understand the data, identify patterns, detect outliers, and explore relationships between features.

## Dataset

* Titanic Dataset

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly

## Steps Performed

* Loaded the Titanic dataset
* Explored dataset information
* Generated summary statistics (mean, median, standard deviation)
* Created histograms for numerical features
* Visualized outliers using boxplots
* Generated a correlation heatmap
* Created a pairplot to analyze feature relationships
* Drew an interactive scatter plot using Plotly
* Identified patterns and feature-level insights

## Output Files

* Task_2_Exploratory_Data_Analysis_(EDA).ipynb
* Titanic-Dataset.csv
* histogram.png
* boxplot.png
* correlation_heatmap.png
* pairplot.png

## Conclusion

EDA helped understand the structure of the Titanic dataset, identify feature relationships, detect outliers, and gain insights useful for machine learning model development.
